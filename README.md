Healthy Food Delivery Website
📌 About Project

This project is a responsive food delivery website created for healthy nutrition programs and daily meal plans.
The website helps users choose meal programs, calculate nutrition goals, explore delivery zones, and learn about the service.

🚀 Features
Responsive Design
Healthy Meal Programs
Delivery Map Section
FAQ Accordion
Nutrition Calculator
Food Menu Cards
Mobile / Tablet / Desktop Support
Modern UI Design
🛠 Technologies Used
HTML5
Tailwind CSS
Flexbox
CSS Grid
